About Smart ASL

Breaking communication barriers with artificial intelligence

Project Overview

Smart ASL Recognition is a deep learning-powered system that translates American Sign Language (ASL) gestures into text in real-time. Using a combination of computer vision and neural networks, the system can recognize 100 different ASL words from live webcam video.

This project aims to bridge the communication gap between the deaf/hard-of-hearing community and those who don't understand sign language, making everyday interactions more accessible.

Technology Stack

Python

Core programming language for backend processing and ML pipeline

TensorFlow / Keras

Deep learning framework for building and training the LSTM model

MediaPipe

Google's framework for hand detection and landmark tracking

MobileNetV2

Lightweight CNN for efficient feature extraction from video frames

OpenCV

Computer vision library for image processing and manipulation

Flask

Lightweight web framework serving the real-time prediction API

Model Architecture

Input

13 consecutive video frames (224x224 pixels)

Feature Extraction

MobileNetV2 produces 1280-dimensional feature vectors per frame

Sequence Model

LSTM layers (256 → 128 units) process temporal patterns

Output

Dense layer with softmax outputs 100 class probabilities

Dataset

The model is trained on the WLASL (Word-Level American Sign Language) dataset, which contains video clips of 100 common ASL words. Each video is preprocessed into sequences of 13 frames, with data augmentation techniques including horizontal flipping, noise injection, and zooming to improve model robustness.

How to Use

  1. Start the Camera: Click the "Start Camera" button on the home page and allow browser access to your webcam.
  2. Position Your Hands: Make sure your hands are visible and well-lit. The system will indicate when it detects your hand.
  3. Perform the Sign: Sign an ASL word. The system needs 13 frames to make a prediction, so hold the sign briefly.
  4. View Results: The predicted word and confidence score appear in real-time on the prediction panel.
  5. Build Sentences: Use the sentence builder to combine predicted words into meaningful phrases, and use text-to-speech to hear them.
